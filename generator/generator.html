Help Directory

done General Notation

Before using this website, we highly advise you to read through this help section, as to not lose progress you put into this tool, and to get informed on how to make a sucessful webstore without error.

Note that there are 6 different sections in the editor that all perform different tasks. These are located on the left of the editor, in the menu. You may return to the home page/miscellaneous pages by clicking on the Webstore Builder logo in the top of the editors menu.

Whenever inputting values in the editor, make sure to click the blue "Submit" button in the selector menu (located in the bottom, and visible when changing properties). Changes will not appear unless the submit button is clicked and the value inputted is valid (the submit button will flash red, signaling if otherwise).

design_services Customize Styling

This section is mainly dedicated to inputting website information, and changing the styles of all elements on your generated webstore.

Initial Settings>Logo: This setting changes your webstores logo throughout the whole page. This requires a url link to the image (meaning your logo needs to be hosted on the web somewhere else, such as imgur.com), this input only excepts links ending in .png, .jpg, .jpeg, or .gif. For example, a logo link should look like this: https://i.imgur.com/BaZjxCC.png.

Initial Settings>Store Name: This setting changes your webstores name throughout the whole page. This requires an input of text (larger than 3 characters).

Header>Header-Border: This setting changes your headers border. With it enabled, a small black line will divide the border from the rest of the page, making it more distinguishable. Disabling it will remove the border line. (Enabled by default)

Header>Header Font: This setting changes your headers font. There are 6 different font options to choose from. (Montserrat by default)

Header>Header Title Color: This setting changes your headers website name (if enabled). This selection only allows for hex codes to be inputted, for example: "#ff0000" is the color red. You can use this tool to find hex codes: https://htmlcolorcodes.com/color-picker/ (#000000 by default)

Header>Header Title: This setting removes or adds the website name to your header. If you have a logo, sometimes it contains the name of your brand inside of it, in which case you can remove the automatically generated one. (Enabled by default)

Body>Body Background Gradient: This setting will change the main body background gradient. You will be prompted with 2 input boxes, each for seperate colors in your gradient. Both input boxes will require a hex value; A tool to find desired hex values can be found in the link above. If you would like a solid color, you may input the same hex value for both boxes, which will result in a solid color background. (#707070 and #ffffff by default)

Body>Body Text: This is the main text shown when first visiting your website, so write something eye catching here. It is suggested to keep this piece of text short, and make your subtext a little more informative.

Body>Body Text Color: This input allows for you to customize the color of the "Body Text" element on your webstore. The input allows only for hex values - refer above for a tool used to generate hex codes. (#000000 by default)

Body>Body Subtext: This is the subtext of the website, located right underneath the primary body text. You can add important details about what your business does, or whatever you want to tell consumers while navigating.

Body>Body Subtext Color: This input allows for you to customize the color of the "Body Subext" element on your webstore. The input allows only for hex values - refer above for a tool used to generate hex codes. (#000000 by default)

Footer>Footer: This selection will enable/disable the footer of the page depending on what is selected. The footer is the area at the very bottom of the page that allows for you to display a little more information on your business, such as where it's located, and/or a link to your social media page. (Enabled by default)

Footer>Address: This option enables/disables the address portion of the footer (if you desire or not to display the location online. (Enabled by default)

Footer>Address Value: You may type the address of your store in this input box. The tool will automatically create a link to Google Maps of the location you entered, and users can be redirected if they click the address in the footer.

Footer>Social Media: This selection shows 4 different options (3 different social media websites, and 1 disable option). Upon selecting one of the social media options, it will change the icon and redirect link shown in the social media portion of the footer. Selecting Disable will not display and social media tags or links. (Enabled by default)

Footer>Social Media Tag: This input is where you can type the tag of your desired social media account (don't include the @ symbol of your tag). The tool will automatically create a link to the social media website you entered, and users can be redirected to that page if they click the tag in the footer.

dashboard_customize Store Properties

This section of the editor is dedicated to the creation of products for your store. Keep reading to learn how to display your products to consumers through your website!

Creating a product: To make a product for your webstore, within the "Store Properties" tab of the editor, there is a button located in the top right, labeled "add Add Another Product". Clicking this button will prompt 3 different input menus sequentially; Refer below for the requirements of each prompt.

Product Name: You may input your products name in this box, it must contain at least 3 characters to be accepted.

Product Image: Your products image should be inserted in this box, this input only excepts links ending in .png, .jpg, .jpeg, or .gif. You can upload images to a seperate online server using a service such as imgur.com.

Product Price: The cost of your product can be inputted in this box. Whatever is inputted must be a positive numerical value, meaning currency symbols will be applied on your webstore automatically. You may use "0" in this box if you want to display a product's price as free.

Upon entering all values into their specified input boxes, a product will be generated on your webstore displaying the name, image, and price.
